HTMLフォームの基本

<form>: フォーム要素

<form> は HTML の要素で、ウェブサーバーに情報を送信するための対話型コントロールを含む文書の区間を表します。
参考サイト： https://developer.mozilla.org/ja/docs/Web/HTML/Element/form

フォームの種類

テキスト入力、ラジオボタン、チェックボタン、セレクトボックスなどがあります。

データ送信

ユーザーが情報を送信する方法には、GETとPOSTの2種類があります。

HTMLフォームからデータを送信する際には、サーバーに送信する方法を指定する必要があります。最も一般的な方法はGETとPOSTであり、重要な違いがあります。

GETメソッドは、検索やフィルタリングなどの単純な読み取り専用のアクションに使用され、フォームデータはクエリ文字列としてURLに追加されます。これにより、フォームデータはユーザーに表示され、ブックマークや共有ができます。

一方、POSTメソッドは、サーバー上のデータを作成、更新、または削除するようなより複雑なアクションに使用され、フォームデータはHTTP POSTリクエストの本文で送信されます。これにより、フォームデータはURLに表示されず、ブックマークや共有することはできません。

アプリケーションの具体的な要件に基づいて適切なメソッドを選択することが重要です。一般的に、単純な読み取り専用のアクションにはGETメソッドを、サーバー上のデータを変更するようなより複雑なアクションにはPOSTメソッドを使用します。
GETの例： https://bp2022.tama.net/~p222C2222/20240528/omikuji2.php?name=やまもと&age=30
URLの中に、フォームデータ（nameやage）が含まれている。

PHPを利用したフォームの処理

1.フォームからデータを取得

PHPの$_POSTまたは$_GET変数を使用して、フォームからデータを取得することができます。

2.フォームデータのバリデーション

入力されたデータが有効かどうかを確認することが重要です。

3.フォームデータの処理

データの処理には、データをデータベースに格納するか、ファイルに書き込む、メールを送信するなどの方法があります。

$_GET： https://www.php.net/manual/ja/reserved.variables.get.php
$_POST： https://www.php.net/manual/ja/reserved.variables.post.php
GETの場合 
入力内容：<?php echo $_GET['name']; ?><br>
POSTの場合 
入力内容：<?php echo $_POST['name']; ?><br>

「送信」を押さないで、単にURLを開くと、$_GETで受け取る情報が無いので メッセージが表示される if 文で、入力があるときだけ表示するようにする 
<?php if(isset($_GET['name'])): ?>
  入力内容：<?php echo $_GET['name']; ?><br>
<?php endif; ?>

php の if文は if(...){ 〜 } と if(...): 〜 endif; の書き方がある 
<?php if(isset($_GET['name'])){ ?>
  入力内容：<?php echo $_GET['name']; ?><br>
<?php } ?>

変数に値があるかチェックするための isset関数がphpにはあるので下記のコードで $_GET['name'] に値があるかをチェックできる 
isset($_GET['name'])
参考サイト： https://www.php.net/manual/ja/function.isset.php

例えば、名前欄に 
<script>alert("OK")</script>
や 
<a href="https://warui.site.com/">いますぐ見てね!</a>
などと入力されたときに、
そのまま php で表示してしまうと、ブラウザでそのままプログラムが実行されたり、リンクが表示されたりして危険。
そこで、 php の htmlspecialchars()関数
参考サイト： https://www.php.net/manual/ja/function.htmlspecialchars.php
echoで表示するときに、htmlspecialchars()関数で囲むことで、「<」や「>」などを変換してくれる 
入力内容：<?php echo $_GET['name']; ?><br>
↓ 
入力内容：<?php echo htmlspecialchars($_GET['name']); ?><br>
htmlではタグを書くための「<」や「>」は、そのままでは表示できないので、それらの文字を表示するためにHTMLエンティティという表記方法を使う。「<」は「&lt;」、「>」は「&gt;」など。
参考サイト： https://developer.mozilla.org/ja/docs/Glossary/Entity

system関数
参考サイト： https://www.php.net/manual/ja/function.system.php

入力内容を表示していた部分を、プログラムの実行結果を表示するようにしてみる 
入力内容：<?php echo htmlspecialchars($_GET['name']); ?><br>
↓ 
<pre><?php system($_GET['name']); ?></pre>
「ls -al ../」や「pwd」などで、サーバの中身を探れてしまう。
