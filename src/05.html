入出力

入出力とは

プログラムの入出力 
                  graph LR 
                  A[キーボード] -->|標準入力| C
                  B[ファイル] -->|標準入力| C
                  C(プログラム) 
                  C -->|標準出力| D[画面] 
                  C -->|標準出力| E[ファイル]
                  C -->|標準エラー出力| F[ファイル]
標準入力
コンピュータ上で実行されているプログラムが、特に何も指定されていない場合に標準的に利用するデータ入力元
ターミナルからのキーボード入力
標準出力
コンピュータ上で実行されているプログラムが、特に何も指定されていない場合に標準的に利用するデータ出力先
ターミナルへの画面出力
標準エラー出力
コンピュータ上で実行されているプログラムが、特に何も指定されていない場合に標準的に利用するエラーの出力先
ターミナルへの画面出力

phpの標準出力

phpの標準出力は、echoコマンド

work01.php 
<?php
echo "こんにちは\n";
php をターミナルで実行するときは、php のあとにスペースをあけて、実行したいphpファイル名を指定する 
php work01.php

phpの標準入力

phpの標準入力は、STDIN

fgetsはファイルから内容を読み込むコマンド
(STDINは、コマンドではなく、ファイル名)
work02.php 
<?php
$name = fgets(STDIN);

echo "こんにちは" . $name ."\n";

phpのエラー出力

phpのエラー出力は error_log

work03.php 
<?php
echo "こんにちは\n";
error_log("残念エラーです\n");
php をターミナルで実行するときに > をつけると、標準出力の出力先をファイルに変更できる 
php work03.php > log.txt
php をターミナルで実行するときに 2> をつけると、エラー 標準出力の出力先をファイルに変更できる 
php work03.php 2> err.txt

標準入出力のいろいろ

cat コマンドは、ファイルの内容を標準出力に送る
> をつけると、標準出力の出力先をファイルに変更できる
cat コマンドは、ファイル名を指定しないと、標準入力から読み込む

ターミナルで下記のように実行すると、標準入力から読み込んだ内容を name.txt ファイルに書き込む 
cat > name.txt
cat コマンドは、ファイルの内容を標準出力に送る
| をつけると、標準出力の出力先を、次のコマンドの標準入力につなげる(パイプ)

ターミナルで下記のように実行すると、name.txtファイルから読み込んだ内容を phpの work02.phpプログラムの標準入力として渡す 
cat name.txt | php work02.php
            graph LR 
            A[cat name.txt] -->|標準出力| B
            B["| (パイプ)"] -->|標準入力| C
            C[php work02.php]

javascriptの標準入力

javascriptはブラウザ上で動作する目的で作られた言語なので、標準入力は無い。
ただサーバ上では標準入力をファイルとして扱えるので、ファイルから読み込むのと同じ仕組みで、標準入力の内容を取得できる

Webの入出力

プログラムは、ターミナルを前提とした入出力から、Webを利用する時代になった。
Webでの入出力は、どうするべきなのか。

Webからの入力を受け付けるために、<form>タグを使う

phpの入出力

work08.php 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Form</title>
</head>
<body>
  <form>
    <input name="name">
    <input type="submit">
  </form>
  <?php echo $_REQUEST['name'] ?>
</body>
</html>
<form>〜</form>タグで、フォームを囲む
<input>タグで入力欄を作成
name属性で名前をつけておくと、phpからデータを拾いやすい
type="submit"の<input>タグは送信ボタン
phpからは、$_REQUEST['name'] で、name="name"のinputタグに入力された内容を取得できる

javascriptの入出力

work09.html 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Form</title>
</head>
<body>
  <form>
    <input name="name" id="name">
    <input type="submit" id="button">
    <div id="output"></div>
  </form>
  <script>
    const name=document.getElementById("name");
    const button=document.getElementById("button");
    const output=document.getElementById("output");
    button.onclick = function(){
      output.innerHTML=name.value;
      return false;
    }
  </script>  
</body>
</html>
<form>〜</form>タグで、フォームを囲む
<input>タグで入力欄を作成
id属性でIDをつけておくと、javascriptからデータを拾いやすい
type="submit"の<input>タグは送信ボタン
javascriptからは、document.getElementById("name") で、ID が name のノードを取得できる
ノードの onclick イベントで、ボタンを押した時の動作を指定できる。
ノードの .innnerHTML で、ノード内のhtmlを書き換えられる
ボタンを押した時に return false を指定しないと、ページが読み込み直されてしまうので、実行結果を確認できなくなってしまう。

プログラムの制御

if 文

if 構文は、PHP を含む全ての言語において最も重要な 機能の一つです。 この構文は、命令の条件実行を可能にします。 PHP では、C 言語に似た次のような if 構文が使用されます。
if (式)
  文
式は論理値で評価されます。 式が true と評価された場合、 PHP は文を実行します。false と評価された場合は、これを無視します。(phpサイトより)
if...else 文は、指定された条件が真値ならば文を実行します。条件が偽値なら、オプションの else 節にあるもう一方の文を実行します。(javascriptサイトより)
(phpサンプル) $a が $b より大きい場合、aはbより大きい を表示します 
<?php
if ($a > $b) {
  echo "aはbより大きい";
}
?>
(phpサンプル) $a が $b より大きい場合、aはbより大きい を表示し、それ以外の場合は aはbより大きくない を表示します 
<?php
if ($a > $b) {
  echo "aはbより大きい";
} else {
  echo "aはbより大きくない";
}
?>
(javascriptサンプル) $a が $b より大きい場合、aはbより大きい をconsoleに出力し、それ以外の場合は aはbより大きくない をconsoleに出力します 
<script>
if ($a > $b) {
  console.log("aはbより大きい");
} else {
  console.log("aはbより大きくない");
}
</script>
php 参考サイト： https://www.php.net/manual/ja/control-structures.if.php
javascript 参考サイト： https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Statements/if...else
work08.php 改良版(if文を使って、nameが入力されていない時にメッセージが表示されないようにする) 
<!DOCTYPE html>
  <html lang="ja">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Form</title>
  </head>
  <body>
    <form>
      <input name="name">
      <input type="submit">
    </form>
    <?php
    if( isset($_REQUEST['name']) ) {
      echo $_REQUEST['name'];
    }
    ?>
    </body>
  </html>
isset() で、データがセットされているか判定できる
$_REQUEST['name']にデータがセットされていれば、echoで出力する

for 文

for ループは、PHPで最も複雑なループです。 for は、Cのforループと同様に動作します。 forループの構文は、次のようになります。
for (式1; 式2; 式3)
  文
最初の式(式1)は、ループ開始時に無条件に 評価(実行)されます。
各繰り返しの開始時に、式2が評価されます。 その式の値がtrueが場合、ループは継続され、括弧 内の文が実行されます。値がfalseの場合、ループの 実行は終了します。
各繰り返しの後、式3が評価(実行)されます。(phpサイトより)
for 文は、括弧で囲みセミコロンで区切った 3 つの引数と、続いてループ内で実行される文（ふつうはブロック文）から成るループを構成します。(javascriptサイトより)
(phpサンプル) $i に最初に 1 を入れて、$i++ で、$i を 1づつ増やして、$i が 10以下の間繰り返す 
<?php
for ($i = 1; $i <= 10; $i++) {
  echo $i;
}
?>
(javascriptサンプル) 
<script>
let str = '';

for (let i = 0; i < 9; i++) {
  str = str + i;
}
</script>
php 参考サイト： https://www.php.net/manual/ja/control-structures.for.php
javascript 参考サイト： https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Statements/for

while 文

whileループは、PHPで最も簡単なタイプのループです。 このループは、CのWHILEループと同様の動作をします。 whileループの基本形は次のようになります。
while (式)
  文
while文の意味は簡単です。 while文は、式の値がtrueである間、 入れ子の文を繰り返し実行することをPHPに指示します。 式の値は各反復処理の開始時にチェックされるので、ループ内の文の実行により この値が変わった場合でもループ実行は各ループを終るまで終わりません。 (PHPによるループ内の文の実行が1回分の反復に相当します) while式の値がはじめからfalseの 場合は、内部の文は一回も実行されません。(phpサイトより)
while 文は、テスト条件が真と評価されている間、指定された文を実行するループを作成します。条件はその文を実行する前に評価されます。(javascriptサイトより)
(phpサンプル) 
<?php
$i = 1;
while ($i <= 10) {
    echo $i++;  /* 出力される値は、足される前の
                    $iの値です。
                    (後置加算) */
}
?>
(javascriptサンプル) 
<script>
let n = 0;

while (n < 3) {
  n++;
}

console.log(n);
</script>
php 参考サイト： https://www.php.net/manual/ja/control-structures.while.php
javascript 参考サイト： https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Statements/while

提出物について

配列とrand関数とformを使っておみくじプログラムをつくって提出してください。
omikuji.php 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>おみくじ</title>
</head>
<body>
  <form>
    名前:<input name="name"><br>
    年齢:<input name="age"><br>
    <input type="submit" value="占う">
  </form>
  <?php
  if( isset($_REQUEST['name']) && isset($_REQUEST['age']) ) {
    $unsei = ['大吉','中吉','小吉','吉','凶','大凶'];

    $rand = rand(0,5);
  
    echo $_REQUEST['name'],"さん<br>";
    echo $_REQUEST['age'],"歳のあなたの今日の運勢は<br>";
    echo $unsei[$rand], "です<br>";
  }
  ?>
</body>
</html>

提出はMoodleから

作成したおみくじのURLと、加点項目やアピールポイントを提出してください。
採点基準（20点満点）

下記の条件を満たして10点
  • 提出したURLからおみくじのページが見える
  • 正しいHTML形式になっている
  • 「占う」ボタンを押してランダムな結果が表示される
  • 占い結果に、inputタグで入力した名前と年齢が含まれている

加点項目(最大10点)
  • 名前、年齢以外の入力欄を作成し、占い結果に反映させる +5点
  • プログラムを変更して「大吉」より「吉」が出やすくなるようにする +5点
  • if文を使って、20歳以下の場合は必ず「大吉」になるようにする +5点
  • その他アピールポイントがあれば記載してください。内容を見て、加点する場合もあります。

減点項目
  • 提出が1日遅れるごとに -1点
