function Kuji(data) {
  this.data=data;

  this.pick = function(){
    let ret = '';
    let total = 0;
    for(let i=0;i<data.length;i++) {
      total += data[i].ritsu;
    }

    const rand = Math.random() * total;
    total=0;
    for(let i=0;i<data.length;i++) {
      ret = data[i].name;
      total += data[i].ritsu;
      if(total > rand) {
        break;
      }
    }

    return ret;
  }
}

JSONとは

JSON

JavaScript Object Notation（JSON、ジェイソン）はデータ記述言語の1つである。
軽量なテキストベースのデータ交換用フォーマットでありプログラミング言語を問わず利用できる。
名称と構文はJavaScriptにおけるオブジェクトの表記法に由来する。
(wikipediaより)
https://developer.mozilla.org/ja/docs/Glossary/JSON
https://ja.wikipedia.org/wiki/JavaScript_Object_Notation

fetch関数

fetch() はグローバルのメソッドで、ネットワークからリソースを取得するプロセスを開始し、レスポンスが利用できるようになったら履行されるプロミスを返します。
https://developer.mozilla.org/ja/docs/Web/API/fetch

tableタグで表を作成
tableタグ： https://developer.mozilla.org/ja/docs/Web/HTML/Element/table
<table>〜</table>
表全体を囲むタグ
<tr>〜</tr>
各行を囲むタグ
<td>〜</td>
各行の中のマス(セル)を囲むタグ
<th>〜</th>
各行の中の見出しのマス(セル)を囲むタグ
<thead>〜</thead>
tableの中のヘッダー部分
<tbody>〜</tbody>
tableの中の本体(body)部分
<tfoot>〜</tfoot>
tableの中のフッター部分
